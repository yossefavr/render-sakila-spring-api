Welcome! This API is connected to a cloud MySQL database. Here is how we deployed it:

Step 1: Database Migration (Aiven)

We exported the Sakila database from a local MySQL instance as a .sql file and imported it into an Aiven.io cloud instance.

Step 2: Environment Variables

Instead of hardcoding passwords, we used variables in application.properties:

spring.datasource.url=${DB_HOST}...

Step 3: Dockerization

Because Render doesn't support Java natively, we created a Dockerfile using Eclipse Temurin to package our Spring Boot app.

Step 4: Cloud Deployment (Render)

We linked our GitHub repo to Render.com, set the runtime to Docker, and added our Aiven credentials as secret Environment Variables.
